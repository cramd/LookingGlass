# LookingGlass — Frontend

Next.js 15 / React / Tailwind frontend for the LookingGlass dashboard.

See the [project README](../README.md) for full setup and architecture documentation.

## Local dev

```bash
npm install
npm run dev   # http://localhost:3000
```

Copy `../env.example` values that start with `NEXT_PUBLIC_` into `.env.local` before running.
